# 977.有序数组的平方

第一个想法：直接先平方后排序

```
#include <vector>
#include <iostream>
using namespace std;

class Solution {
public:
    vector<int> sortedSquares(vector<int>& nums) {
        int k;
        int i=0;
        int length =nums.size();
        for(i=0;i<length;i++){   //遍历数组
            nums[i]=nums[i]*nums[i];
        }

        for(k=length;k!=1;k--){
            for(i=0;i<k-1;i++){
                if(nums[i]>nums[i+1]){
                int j=nums[i];
                nums[i]=nums[i+1];
                nums[i+1]=j;
                }
            } 
        }

        return nums;
    }
};
```

这个算法理论上来说是可以的，但是算法时间复杂度太高，所以力扣不通过，后续优化一下。
