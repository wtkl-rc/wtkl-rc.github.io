# sort

\#include < algorithm >

sort (start, end, cmp)

start表示排序数组的起始位置

end表示数组结束地址的下一位

cmp用于规定排序的方法，默认升序

```
sort(nums.begin(), nums.end()); // 快速排序nums是个vector
```
