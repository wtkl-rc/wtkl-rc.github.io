# 冒泡排序

```
int main (){
    int a[5]={3,5,2,1,7};
    int length=sizeof(a)/sizeof(int);  //length=5
    int i;
    
    for(i=length;i>1;i--){            //n个数对比n-1次 (1,5]
        for(int j=0;j<i-1;j++){       //对比i-1次
            if(a[j]>a[j+1]){          //进行交换 
                int k=a[j+1];
                a[j+1]=a[j];
                a[j]=k;
            }
        }
    }        
    system("pause");
    return 0;
}
```

总结：先确定对比几次
